# はじめに

# 勾配降下法

- 目的関数が最小となるパラメータを勾配表によって探す。
- 目的関数が各パラメータによって微分できる必要がある。

# 確率的勾配降下法

- 無作為に選び出されたデータを用いて行う勾配降下法である。

# ミニバッチ勾配降下法

- 無作為に選び出された複数のデータを用いて行う勾配降下法である。
